# 중기부 온라인 모니터링 판단 프로그램

기사 링크 하나만 넣으면, 본문을 읽고 "수집(O) / 제외(X)"를 판단해주는 CLI 도구입니다.

## 1. 설치

```bash
cd judge_article
pip install -r requirements.txt
```

## 2. API 키 발급 (아직 없다면)

1. https://console.anthropic.com 접속 후 계정 생성/로그인
2. 결제수단 등록 (API는 claude.ai 구독과 별개로 사용량만큼 과금됩니다)
3. 좌측 메뉴에서 **API Keys → Create Key**
4. 발급받은 키(`sk-ant-...`)를 복사

## 3. 키 등록

이 폴더에 `.env` 파일을 만들고 아래처럼 한 줄만 넣으세요.

```
ANTHROPIC_API_KEY=sk-ant-여기에-발급받은-키-붙여넣기
```

## 4. 실행

```bash
python judge.py "https://mbnmoney.mbn.co.kr/news/view?news_no=MM1005880103"
```

출력 예시:

```
==================================================
매체 : 매일경제TV
제목 : "악성 리뷰 남길까 속이 타요"…네이버 별점 부활에 불안한 사장님들
--------------------------------------------------
판단 : ⭕ 수집
주제 : 네이버 별점 재도입에 따른 소상공인 피해 우려
근거 : 네이버 별점 부활로 인한 소상공인 부담을 다룬 기사로, 수집 기준에 해당함.
==================================================
```

## 5. 여러 링크 한 번에 처리하기 (배치)

링크를 하나씩 넣기 귀찮으면 `judge_batch.py`를 쓰세요.

**방법 A - 파일에 붙여넣기 (링크 많을 때 추천)**

`links.txt`를 열어서 링크를 한 줄에 하나씩 쫘라락 붙여넣고 저장한 뒤:

```bash
python judge_batch.py links.txt
```

**방법 B - 터미널에 바로 붙여넣기**

```bash
python judge_batch.py
```
링크를 한 줄에 하나씩 붙여넣고, 다 넣었으면 Mac/Linux는 `Ctrl+D`, Windows는 `Ctrl+Z` 후 `Enter`.

두 방법 다 실행하면:
- 화면에 하나씩 처리되는 대로 ⭕/❌ 표시
- 마지막에 전체 요약 (수집 몇 건 / 제외 몇 건)
- 같은 폴더에 `결과_날짜시간.csv` 파일로 자동 저장 (엑셀로 바로 열림, 담당자한테 제출할 때 이거 쓰면 됨)

기본으로 5개씩 동시에 처리합니다. 링크가 아주 많거나(수십~수백 개) API 요청 제한 에러가 뜨면
`judge_batch.py` 상단의 `MAX_WORKERS` 값을 3 정도로 낮추세요.

## 6. 판단 기준 수정하기

담당자 피드백이 새로 오면 `criteria.md` 파일만 수정하세요. 코드는 건드릴 필요 없습니다.
파일을 저장하면 다음 실행부터 (단일/배치 둘 다) 바로 반영됩니다.

## 참고 / 한계

- 로그인이 필요하거나 자바스크립트로 본문을 그리는 사이트는 본문 추출이 실패할 수 있습니다.
  이런 경우 에러 메시지가 뜨며, 필요하면 사이트별 추출 로직을 추가해드릴 수 있습니다.
- 기본 모델은 `claude-sonnet-5` 입니다. 판단량이 많아 비용이 부담되면 `common.py`의
  `MODEL` 값을 `claude-haiku-4-5-20251001` 로 바꾸면 더 저렴하고 빠릅니다 (정확도는 약간 낮아질 수 있음).
  이 값을 바꾸면 `judge.py`, `judge_batch.py` 둘 다에 자동으로 적용됩니다.
